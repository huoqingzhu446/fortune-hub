## 1. 总体分工结论

P0 里不依赖外部账号的工程项已由 Codex 先行处理，包括工程安全收口、生产配置校验、模拟能力下线、迁移流程固化、隐私删除申请入口、情绪自检合规提示、发布态检查接口和小程序码 path 加固。

必须由项目方配合的是微信支付商户资料、微信小程序后台配置、正式域名 / 证书 / 备案、隐私与用户协议文案、情绪自检合规口径、正式运营数据和生产环境密钥。

建议执行顺序：

1. 先收集外部发布资料：微信支付、合法域名、隐私协议、短信、正式内容。
2. 在 staging 跑 migration、导入正式内容，然后调用发布态检查接口确认无 fail。
3. 拿到外部资料后做真实支付、真机联调、体验版验收和正式审核。
